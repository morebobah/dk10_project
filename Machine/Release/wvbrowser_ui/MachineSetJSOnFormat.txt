00. MACHINE - Содержит список машин
01. Список машин, все имена начинающиеся на x нужны только для программы.
01. xNAME - Название машины
01. NUMBER - номер машины
01. xDESCRIPTION - описание машины
01. PINS - содержит список исполнителей
01. xGROUPS - содержит список групп исполнителей
02. Список исполнителей
02. PINTYPE - перечисляемый элемент: M, S, W указывает на то какой тип исполнителя
02. HWObj - указывает адрес исполнителя по 485 интерфейсу
02. ADR - номер использованного порта
02. xGR - группа к которой принадлежит исполнитель
03. Список групп исполнителей
03. xGN - имя группы
03. xGD - описание группы

Передаваемые сообщения на контроллер:
1. status - запрашиваем текущий статус исполнителей
2. getms - запрашиваем текущую конфигурацию машин
3. setms - запрашиваем готовность загрузить конфитгурацию
4. setcommand - отправляем запрос на получение команды
5. setpacket - отправляем запрос на запись автоматичекой команды

Сообщения получаемые с контроллера
1. Ответ на запрос status получаем json со статусами
2. Ответ на getms получаем json с машинами
3. Ответ на setms получаем ready и можем отправлять json с машинами
4. Ответ на setcommand получаем ready и можем отправлять json с командами
5. Ответ на setpacket получаем ready и можем отправлять json со списком команд для записи.
