Подключение и запуск станка

Перед запуском станка убедитесь в корректности соединений, согласно электрической схеме. Подайте питание на станок. Роутер входящий в состав станка должен быть включен. SD какрта с программами станка должна быть установлена в модуль контроллера. Если все условия выполнены правильно, то индикатор рисунок 1 должен показывать ход загрузки контроллера.
Запускать станок в ручном и автоматическом режимах позволяет приложение Machine для операционной системы Windows.
Запускать станок в автоматическом режиме возможно с помощью приложения BricksFactory, которое должно быть для этого установлено на мобиьлное устройство под управлением операционной системы android.
Порядок получения доступа к элементам управления станком с помощью программы Machine следующий:
  • Включить питание станка убедится в том, что он запущен.
  • Считать с экрана индикатора, рисунок 1, значение IP адреса выданного роутером контроллеру станка.
  • Записать значение IP адреса в соответствующее поле, рисунок 2.
  • Нажать кнопку соединения с контроллером.
  • Получить доступ к закладкам с элементами управления станком, рисунок 3.

Аварийная остановка

Контроллер запускает программы автоматического или ручного управления по внутренним алгоритмам, которые могут приводить к ошибкам или авариным ситуациям. При возникновении аварийной ситуаций вы можете остановить выполнение любой автоматической программы либо ручной операции с помощью нажатия кнопки Esc. Нажатие этой кнопки приводит к блокировки контроллера на 5 минут. Рекомендовано, при использовании аварийной остановки программы незамедлительно отключить питание от контроллера!

Ручное управление, включение и выключение двигателей

Программа Machine позволяет запустить или выключить любой из двигателей станка, который подключен к контроллеру и прописан в конфигурации. Конфигурация станка хранится на SD карте контроллера. Ручное управление осуществляется на закладке "Ручной режим", который становится доступным после того, как программа была присоединена к контроллеру. Общий вид элементов управления изображен на рисунке 6. В левой части окна размещена панель gCode в которой записываются коды управления станком.
В правой части размещены панели с элементами управления станком.
На панели инструментов кажды элемент станка имеет свое пиктографическое изображение:
  • - мотор
  • - концевой выключатель
  • - весы
  • - таймер
Каждая пиктограмма объеденена с элементами управления:
  • - Старт. Запуск двигателя либо срабатывание концевого выключателя, либо проверка веса, либо запуск таймера.
  • - Стоп. Остановка двигателя.
С помощью элементов управления можно выполнить следующие ручные или полуавтоматические операции:
Запуск двигателя
Осуществляется нажатием элемента управления возле соответствующей пиктограммы .
Остановка двигателя
Осуществляется нажатием элемента управления возле соответствующей пиктограммы .
Работа двигателя по таймеру.
  • Выполняется команда запуск двигателя.
  • Устанавливается таймер, возле пиктограммы записывается количество секунд, в течении которых должен работать двигатель. Нажимается элемент управления таймера . Запущенный таймер загорается красным цветом.
  • Выполняется команда остановка двигателя, при этом таймер перестает быть отмеченным.
Работа двигателя до срабатывания концевого выключателя.
  • Выполняется команда запуск двигателя.
  • Устанавливается концевой выключатель, возле пиктограммы нажимается элемент управления сенсора . Запущенный концевой выключатель загорается красным цветом.
  • Выполняется команда остановка двигателя, при этом таймер перестает быть отмеченным.
Работа двигателя до набора определенного веса.
  • Выполняется команда запуск двигателя.
  • Устанавливается вес на весах, возле пиктограммы записывается количество килограмм которое необходимо отмерить, нажимается элемент управления весов . Запущенные весы загораются красным цветом.
  • Выполняется команда остановка двигателя, при этом таймер перестает быть отмеченным.
В ручном режиме можно работать в двух режимах:
  1. Непосредственная отправка команд на станок.
  2. Запись команд с последующей отправкой на станок либо записью в качестве автоматической программы.
Переключение между режимами осущствляется с помощью ползункового переключателя расположенного на центральной панели управления, рисунок 7.

Автоматические программы станка

Автоматические программы записываются на SD карту подключенную к контроллеру. Запуск автоматических программ может осуществляться с компьютера программой Machine или с мобильного устройства с помощью приложения BricksFactory. Запуск автоматических программ в приложение Machine осуществляется на закладке "Автоматический режим".
